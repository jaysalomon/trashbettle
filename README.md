## 🎯 **Key Innovation**

This research presents a breakthrough approach to autonomous robotics through **multi-functional flow lattice architecture** - a unified system where the same channels serve as:
- ⚡ **Energy generation network** (distributed micro-combustion)
- 🏗️ **Structural skeleton** (load-bearing lattice)
- 🌡️ **Thermal management** (heat delivery system)
- 🧠 **Control pathways** (distributed processing)

## 🚀 **Quick Start**

### 1. Clone the Repository
```bash
git clone https://github.com/jaysalomon/trashbettle.git
cd trashbettle
```

### 2. Set Up Environment
```bash
# Install dependencies
pip install -r requirements.txt

# Verify CUDA for GPU simulations
python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}')"
```

### 3. Run Simulations
```bash
# Heat transfer analysis
python simulations/scripts/heat_solver.py

# Pressure drop validation  
python simulations/scripts/pressure_drop.py

# Full system simulation
python simulations/scripts/full_system_sim.py
```

### 4. Generate Paper
```bash
cd paper/
make  # Compiles LaTeX to PDF
```
